# Fake Store API Automation

## Operations in Postman
- Create User
- Create Product
- Add Product to Cart
- Update Cart
- Delete Cart, product, user

## Steps

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
3. Run the tests:
   ```bash
    npm run fakeapi
 

4. View the HTML report in:
   **reports/fake-store-report.html**
